Text Viewer Window

In the text viewer window the follwing shortcuts can be used:

  • UpArrow - Moves the viewport one line up.
  • DownArrow - Moves the viewport one line down.
  • LeftArrow - Moves the viewport a few characters to the left.
  • RightArrow - Moves the viewport a few character to the right.
  • PageUp - Moves the viewport one (viewport) page up.
  • PageDown - Moves the viewport one (viewport) page down.
  • Home - Moves the viewport to the beginning of the text.
  • End - Moves the viewport to the end of the text.
  • Alt+Wheel - Scrolls the page horizontally (the Wheel is the mouse wheel).
  • Ctrl+Wheel - Zooms the text.
  • Ctrl++ - Zoom the text in (increase the font size by one point).
  • Ctrl+- - Zoom the text out (decrease the font size by one point).
  • Ctrl+A - Selects all text.
  • Ctrl+C - Copy selected text to the clipboard.
  • Ctrl+F - Open the find text dialog.
  • Ctrl+S - Open the file save dialog.
  • Ctrl+W - Toggle wrapping text to the width of the window.
